## Database

- Current ORM is Prisma

> `prisma migrate` is only for SQL Database

> Run `npx prisma generate` instead for Prisma client typesafe

> For prod use `npx prisma db push` Read the warning before pushing

## Permission

| Role    | Permissions                 |
| ------- | --------------------------- |
| USER    | - Read own 'usersData'      |
|         | - Read 'productsData'       |
| MANAGER | - Extend to Role.USER       |
|         | - Read any 'userData'       |
|         | - Read any 'productsData'   |
| ADMIN   | - Extend to Role.MANAGER    |
|         | - Update any 'userData'     |
|         | - Delete any 'userData'     |
|         | - Create any 'productsData' |
|         | - Update any 'productsData' |
|         | - Delete any 'productsData' |
| OWNER   | - Extend to Role.ADMIN      |
|         | - Create any 'userData'     |

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
